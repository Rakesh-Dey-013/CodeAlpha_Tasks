## ğŸ¨ Customization

### Updating Content

1. **Personal Information**: Edit `index.html` to update your name, bio, and contact details
2. **Styling**: Modify CSS files in the `src/` directory:
   - `styles.css`: Global styles and variables
   - `about.css`: About section specific styles
   - `card.css`: Card components
   - `service.css`: Services section styles

3. **Images**: Replace images in the `Assets/` folder with your own

### Particle.js Configuration

Edit the Particle.js settings in `script.js` to customize:
- Particle count
- Colors
- Movement speed
- Interaction effects
